O ng-template é uma diretiva que permite criar um template reutilizável. Ele não é renderizado no DOM até que seja chamado explicitamente.

Aqui está um exemplo de texto que será renderizado quando o ng-template for chamado. O ng-template é útil para criar templates reutilizáveis e dinâmicos. Você pode usar o ng-template para criar componentes personalizados ou para renderizar conteúdo condicionalmente, pelo if else.



ng-container

O ng-container é um elemento que não é renderizado no DOM, mas permite agrupar elementos e aplicar diretivas a eles. Isso é útil quando você deseja aplicar uma diretiva a um grupo de elementos sem adicionar um elemento extra ao DOM.

ng-content

O ng-content é uma diretiva que permite projetar conteúdo em um componente filho. Isso significa que você pode criar um componente pai e projetar conteúdo dentro dele, permitindo a reutilização de componentes.
