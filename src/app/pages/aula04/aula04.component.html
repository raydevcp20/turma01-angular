Decorator @input e @output

@input

Usado principalmente para enviar dados de um componente pai para um componente filho.

Abaixo foi criado um componente de bot達o e um componente de input externos.

O bot達o recebe um texto e uma cor.



O input recebe um placeholder, um valor e um status.



@output

Usado principalmente para enviar dados de um componente filho para um componente pai.

Abaixo foi reutilizado os componentes de bot達o e input externos.

O bot達o envia um evento de click.


O input envia um evento de focus.
