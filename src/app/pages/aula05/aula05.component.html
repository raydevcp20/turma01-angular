Decorator @input e @output

@input

Usado principalmente para enviar dados de um componente pai para um componente filho.

Abaixo foi criado um componente de botão e um componente de input externos.

O botão recebe um texto e uma cor.



O input recebe um placeholder, um valor e um status.




@output

Usado principalmente para enviar dados de um componente filho para um componente pai.

Abaixo foi reutilizado os componentes de botão e input externos.

O botão envia um evento de click.


O input envia um evento de focus.




Input Signals

input()

Forma mais atual de receber dados de um componente pai para um componente filho.

Abaixo foi reutilizado os componentes de botão e input externos porem com o input signal.

O botão envia um evento de click.


O input envia um evento de focus.
